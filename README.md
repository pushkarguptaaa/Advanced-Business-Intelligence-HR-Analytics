# 📊 HR Analytics – Data Modeling & Executive Reporting

## 📌 Overview

This project focuses on building a comprehensive **HR analytics dashboard** in Power BI to monitor workforce metrics such as **headcount, employee retention, and turnover**.

The objective is to transform raw HR data into a structured data model and deliver **interactive, KPI-driven insights** for executive decision-making.

---

## 🎯 Objectives

- Analyze workforce composition and trends  
- Measure employee retention and turnover  
- Design a scalable data model for HR reporting  
- Build an interactive dashboard for business users  

---

## 🛠️ Tools & Technologies

- **Power BI**
  - Power Query (Data Transformation)
  - Data Modeling (Star Schema)
  - DAX (Measures & KPIs)
- **Excel / CSV** (Data Source)

---

## 🧱 Data Modeling

- Performed data exploration and cleaning using **Power Query**  
- Designed a **star schema** with:
  - Fact table: Employee records  
  - Dimension tables: Date, Department, Role, Location  
- Established relationships for efficient filtering and analysis  

---

## 📐 Key Metrics & Calculations

### 👥 Headcount
- Total active employees over time  
- Department-wise and role-wise distribution  

### 🔁 Retention Rate
- Percentage of employees retained over a period  
- Time-based retention trends  

### 🔻 Turnover Rate
- Employee exits over time  
- Monthly and yearly turnover analysis  

### 📊 DAX Implementation
- Time intelligence calculations  
- Dynamic KPI measures  
- Aggregations and filtering logic
